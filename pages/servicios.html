Minecraft
Servidor local de Minecraft implementado como entorno educativo interactivo dentro del proyecto Servenium. Permite a los estudiantes conectarse en red local para experimentar conceptos reales de hosting, administración de servidores, optimización de recursos y gestión de usuarios, demostrando cómo un servicio de entretenimiento puede convertirse en una herramienta práctica de aprendizaje tecnológico.
Streaming
Plataforma de streaming multimedia local diseñada para compartir contenido educativo y audiovisual sin depender de servicios externos en internet. Utilizando tecnologías open source, el sistema permite gestionar bibliotecas multimedia, controlar accesos y demostrar el funcionamiento real de servidores de distribución de contenido dentro de una infraestructura doméstica optimizada.
Website
Sitio web oficial del proyecto Servenium alojado directamente en el servidor local. Funciona como plataforma informativa del homelab y evidencia práctica del despliegue de servicios web reales mediante tecnologías como contenedores, servidores HTTP y administración de red local, integrando documentación, presentación académica y demostración funcional del proyecto educativo InnovaServer EDU.
