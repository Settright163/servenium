CasaOS
Sistema base del homelab que centraliza la gestión de servicios, contenedores y almacenamiento mediante una interfaz web sencilla.
Crafty
Panel de administración para servidores Minecraft que permite controlar usuarios, recursos y estados del servidor desde navegador.
Docker
Tecnología de contenedores utilizada para aislar servicios, facilitar despliegues rápidos y mantener la arquitectura modular.
Jellyfin
Servidor multimedia autoalojado que organiza películas y series permitiendo streaming local sin depender de servicios externos.
Jellyseerr
Interfaz de solicitudes multimedia que automatiza peticiones de contenido integrándose con el sistema de descarga y gestión.
Linux Mint
Sistema operativo anfitrión del homelab, elegido por su estabilidad, eficiencia y compatibilidad con servicios de servidor.
NGINX
Proxy reverso encargado de enrutar solicitudes hacia cada servicio interno usando dominios locales dentro de la red.
Prowlarr
Administrador central de indexadores que conecta motores de búsqueda con Radarr y Sonarr para automatizar descargas.
Qbittorrent
Cliente de descargas automatizado que recibe tareas desde los gestores multimedia y administra transferencias.
Radarr
Sistema de gestión automática de películas que busca, descarga y organiza contenido dentro del servidor multimedia.
Sonarr
Gestor automatizado de series que monitorea episodios nuevos y mantiene organizada la biblioteca multimedia del homelab.
